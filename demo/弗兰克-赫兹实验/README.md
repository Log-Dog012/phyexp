豆包：

✅ 这台 **Agilent DSO-X 2012A 示波器** 支持**快速傅里叶变换（FFT）**，可以用来做频域分析。

关键依据
- 官方资料和技术文档明确指出，该型号属于 InfiniiVision 2000 X 系列，内置 **FFT 功能**，属于其波形数学运算的一部分。
- 你可以通过面板上的 `Math`（数学运算）菜单，选择 FFT 功能，将时域信号转换为频域频谱，用于分析信号的频率成分、谐波失真等。

操作提示
1.  捕获时域波形后，按下 `Math` 键。
2.  在数学函数菜单中选择 `FFT`。
3.  选择要分析的通道，调整窗函数（如汉宁窗、矩形窗）和频率范围，即可查看频谱图。
